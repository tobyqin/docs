## 本地运行

1. 安装 Hugo
   ```bash
   # Mac
   brew install hugo
   
   # Windows
   choco install hugo
   ```

2. 克隆仓库
   ```bash
   git clone https://github.com/tobyqin/docs.git
   cd docs
   ```

3. 启动本地服务器
   ```bash
   hugo server -D
   ```

4. 访问 <http://localhost:1313/docs>
